# Support

## Getting Help

Use GitHub issues for:

- reproducible SDK bugs
- documentation problems
- feature requests
- benchmark reproducibility questions

Before opening an issue, please check:

- [README.md](README.md)
- [docs/SDK_CONTRACTS.md](docs/SDK_CONTRACTS.md)
- [docs/BENCHMARK_EVIDENCE.md](docs/BENCHMARK_EVIDENCE.md)
- [benchmarks/README.md](benchmarks/README.md)

## Useful Issue Details

Include:

- SDK version or commit SHA
- Python version and operating system
- install command and optional extras
- minimal reproduction code
- full exception text
- whether the issue involves Label Studio, managed Docker, benchmarks, caches, or persisted state

## Security

Security reports should follow [SECURITY.md](SECURITY.md), not public issues.
