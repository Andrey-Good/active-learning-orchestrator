# Agent Workspace

This directory stores repository-local agent guidance and task history that is useful for future maintainers and AI coding agents.

Tracked content:

- `tasks/`: task documents, review notes, and implementation/review plans created during development.

Ignored local content:

- `tmp/`: scratch workdirs, virtual environments, generated runs, and temporary stress-test artifacts.
- `archive/`: local archival copies of files moved during release cleanup.

Do not commit secrets, API tokens, local datasets, virtual environments, generated benchmark runs, or runtime workdirs here.
